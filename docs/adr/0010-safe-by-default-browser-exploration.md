# 瀏覽器探索預設封鎖副作用與外部網路

Playwright 自動探索預設只執行頁面導覽、開關浮層、切換分頁、展開收合及無副作用篩選等可逆操作；刪除、付款、發布、寄送、登出、上傳與不明表單提交均禁止。來源專案的準備階段只允許連線探索設定 allowlist 的套件 registry，優先使用 lockfile 與 frozen 安裝，私有憑證只由 Docker secret 或環境變數提供；依賴可保存於獨立 cache volume。安裝完成後，網站在隔離網路中執行，預設只能存取 Compose 內部服務，外部 API 以 mock／stub 及探索設定中的回應資料取代。只有探索設定逐項 allowlist 的操作或 host 才能開放，不提供全域允許選項；不確定的互動標記為未探索並保留證據。
