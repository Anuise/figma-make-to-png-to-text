# 以兩個來源專案及來源更新驗證第一版

第一版須在 Windows 11、Docker Desktop WSL2 與單一 Docker Compose 中完成端到端驗收：從唯讀父目錄選擇 Figma Make React 來源專案，建立工作副本並啟動網站，在安全與預算限制內探索 Desktop 畫面，經人工確認後由 Antigravity SDK 分階段產生並確認工作流、工程工項、API、資料模型與工時，且支援取消、續跑、局部重跑、持久狀態、詳細 log 與可攜 ZIP 交付。驗收需使用第二個來源專案證明流程未針對單一專案寫死，並用更新後來源版本證明歷史保留、差異追蹤與 checkpoint 生效。第一版不承諾 macOS／Linux、多人協作、無限制探索或自動商業優先級。
